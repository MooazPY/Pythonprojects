# 🎨 The Hirst Painting

A Python Turtle generative art project inspired by **Damien Hirst's** iconic spot paintings. The script draws a 10×10 grid of colorful dots using a curated palette of RGB colors.

---

## ✨ Features

- 10×10 grid of randomly colored dots
- Hand-picked RGB color palette of 27 colors inspired by real Hirst paintings
- Fully generative — a different arrangement every run
- Clean output with no pen lines between dots

---

## 🚀 Getting Started

### Prerequisites

- Python 3.x (Turtle is built-in, no extra installs needed)

### Run the script

```bash
python The_Hirst_Painting.py
```

Click anywhere on the screen to close the window.

---

## ⚙️ How It Works

1. A turtle starts at the bottom-left corner of the canvas
2. It draws 10 dots across, then moves up a row and repeats
3. Each dot picks a random color from the curated RGB palette
4. After 100 dots, the turtle hides and the painting is complete

---

## 🎨 Color Palette

The palette includes 27 carefully selected RGB colors such as:

- Sky blue `(80, 177, 230)`
- Coral red `(238, 48, 84)`
- Mint green `(121, 226, 187)`
- Soft yellow `(229, 225, 97)`
- Deep navy `(54, 80, 159)`
- and many more...
