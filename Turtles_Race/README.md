# 🐢 Turtles Race

A fun Python Turtle graphics game where 6 colorful turtles race across the screen. Place your bet on a color and see if your turtle wins!

---

## ✨ Features

- 6 turtles racing in different colors
- Player prediction via a pop-up dialog before the race
- Random movement speed for each turtle each step
- Win/loss result printed based on your prediction

---

## 🚀 Getting Started

### Prerequisites

- Python 3.x (Turtle is built-in, no extra installs needed)

### Run the game

```bash
python Turtles_Race.py
```

---

## 🖥️ Gameplay

1. A dialog box appears asking you to predict the winner
2. Type one of the available colors: `red`, `orange`, `yellow`, `green`, `purple`, `blue`
3. The race begins — each turtle moves forward by a random amount each step
4. The first turtle to cross the finish line wins
5. The result is printed in the terminal

```
The winner is the green turtle
Your prediction is right.
```
