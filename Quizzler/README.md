# 🧠 Quizzler App (Tkinter)

A simple GUI-based quiz application built with Python and Tkinter. Users can sign up, answer True/False questions, and track their score in real-time.

---

## ✨ Features

- User sign-up system (email & password stored locally)
- Interactive quiz interface with GUI
- True/False question system
- Real-time score tracking
- Automatic question progression
- End-of-quiz handling

---

## 🚀 Getting Started

### Prerequisites

- Python 3.x

### Run the app

```bash
python main.py
```

---

## 🖥️ Gameplay

```
1. Enter your email and password in the sign-up window
2. Click "Sign Up" to save your credentials
3. Quiz window opens automatically
4. Read the question displayed
5. Click:
   - ✔ True
   - ✖ False
6. Score updates based on correct answers
7. App closes automatically when quiz ends
```

---

## ⚙️ How It Works

1. User enters email and password in the first window  
2. Credentials are saved locally in a text file  
3. Quiz window launches using Tkinter GUI  
4. Questions are fetched from a question bank  
5. User answers using True/False buttons  
6. Score updates dynamically  
7. App ends when all questions are completed
